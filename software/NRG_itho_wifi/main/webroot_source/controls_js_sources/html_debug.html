I2C extra debug functionality:

Only use this feature (briefly) for debugging issues. This option enables an extra menu where more I2C options and logging is available.
Enabling the I2C debug menu will also halt I2C bus activity in case of errors to be able to save logging!



I2C sniffer:

Only use this feature (briefly) for debugging issues. The I2C sniffer works only on sniffer capable devices (non-CVE; all versions, CVE; as of hw rev. 2.5).
See system log for confirmation.
Sniffer output is available through syslog. (reboot needed)
